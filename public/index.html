Text to Speech

The Text to Speech service understands text and natural language to generate synthesized audio output complete with appropriate cadence and intonation. It is available in 11 voices across 5 languages, including the voice used by Watson in the 2011 Jeopardy match.

Resources:

API Overview Documentation Fork on Github

Input Text

The text language must match the selected voice language: Mixing language (English text with a Spanish male voice) does not produce valid results. The synthesized audio is streamed to the client as it is being produced, using the HTTP chunked encoding. The audio is returned in the Ogg Vorbis format which can be played using VLC and Audacity players.
